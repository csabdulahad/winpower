# WinPower
A bundle of cmd/powershell tools, that powers you.

This repository hosts a collection of cmd/powershell tools which can greatly simplies you day to day tasks in CMD or Powershell.

# Installation
Being a command line or terminal level tool, winpower can be installed in many ways. However, we are presenting our recommened way as it provides maximum accessiblilty to tool set.

1. Download the repository from the github & extract the zip file.
2. Place the winpower folder inside Program Files folder.
3. Add the folder path to the system level environment.

Note: In order to add the path, for windows you can do the following.

1. In Start menu search "environment variables".
2. Open the suggested program, then & click Environment Variables button.
3. Under System Variables find Path variable look for path variable.
4. Double click on the variable to add the winpower path.
